## 1. การเข้าสู่ระบบ

### วิธีเข้าสู่ระบบ

1. เปิดเบราว์เซอร์ ไปที่ `http://admin.farm.localhost`
2. กรอก **Username** และ **Password** ที่ได้รับจาก Owner
3. คลิก **"เข้าสู่ระบบ"**

> ⚠️ หากไม่สามารถเข้าสู่ระบบได้ กรุณาติดต่อ Owner เพื่อตรวจสอบสิทธิ์การเข้าถึง

## 2. Dashboard - ดูภาพรวมฟาร์ม

### 2.1 หน้า Dashboard หลัก

เมื่อเข้าสู่ระบบ คุณจะเห็น Dashboard ที่แสดงภาพรวมของฟาร์ม:

| วิดเจ็ต | ข้อมูลที่แสดง |
|--------|--------------|
| **Stats Cards** | จำนวนงานทั้งหมด, เสร็จแล้ว, กำลังทำ, รอดำเนินการ |
| **สถานะพืช** | จำนวนต้นไม้แยกตามสถานะ (ดี/เฝ้าระวัง/มีปัญหา) |
| **พนักงาน** | ภาระงานของพนักงานแต่ละคน |
| **รายงานปัญหา** | ปัญหาล่าสุดที่ต้องจัดการ |
| **งานวันนี้** | ความคืบหน้างานวันนี้ |
| **สภาพอากาศ** | อุณหภูมิและความชื้น (เชื่อมต่อ API ภายนอก) |

### 2.2 กรองข้อมูลตามช่วงเวลา

| ตัวเลือก | ความหมาย |
|----------|----------|
| **วันนี้** | แสดงข้อมูลวันปัจจุบัน |
| **สัปดาห์นี้** | แสดงข้อมูลสัปดาห์ปัจจุบัน |
| **เดือนนี้** | แสดงข้อมูลเดือนปัจจุบัน |
| **กำหนดเอง** | เลือกวันที่เองได้ |

## 3. สร้าง/มอบหมายงาน

### 3.1 สร้างงานใหม่

**ขั้นตอนที่ 1:** คลิก **"+ สร้างงานใหม่"** หรือไปที่ **"งาน" → "สร้างงาน"**

**ขั้นตอนที่ 2:** กรอกรายละเอียดงาน:

| ช่อง | คำอธิบาย |
|------|----------|
| **ชื่องาน** | ชื่อที่สื่อความหมายของงาน |
| **ประเภทงาน** | 💧 รดน้ำ, 🧪 ใส่ปุ๋ย, ✂️ ตัดแต่ง, 🔬 ตรวจสอบ, 🐛 กำจัดแมลง, 📦 อื่นๆ |
| **แปลง/พืช** | เลือกแปลงหรือพืชที่ต้องทำงาน |
| **มอบหมายให้** | เลือกพนักงานที่รับผิดชอบ |
| **วันที่เริ่ม-สิ้นสุด** | กำหนดเวลาเริ่มและครบกำหนด |
| **รายละเอียด** | คำอธิบายงานเพิ่มเติม |

### 3.2 มอบหมายงานให้พนักงานหลายคน

หากต้องการมอบหมายงานเดียวกันให้พนักงานหลายคน:

1. สร้างงานตามปกติ
2. ที่ช่อง **"มอบหมายให้"** กด **Ctrl + คลิก** (Windows) หรือ **Cmd + คลิก** (Mac) เพื่อเลือกหลายราย
3. หรือคลิกที่ **"เลือกทั้งหมด"** หากต้องการมอบหมายให้พนักงานทุกคน

### 3.3 จัดการงานที่มอบหมายแล้ว

| การดำเนินการ | วิธีทำ |
|--------------|--------|
| **แก้ไขงาน** | ไปที่รายการงาน → คลิกงาน → คลิก "แก้ไข" |
| **ยกเลิกงาน** | เปิดรายละเอียด → เมนู ⋮ → "ยกเลิกงาน" → ใส่เหตุผล → ยืนยัน |
| **ลบงาน** | ไม่แนะนำ - ใช้ "ยกเลิก" แทน เพื่อเก็บประวัติ |

### 3.4 ติดตามสถานะงาน

| สถานะ | ความหมาย | การดำเนินการ |
|-------|----------|--------------|
| ⏳ รอดำเนินการ | พนักงานยังไม่เริ่มทำ | ติดตามว่าพนักงานเห็นงานหรือยัง |
| 🔄 กำลังทำ | พนักงานกำลังดำเนินการ | รอรายงานผล |
| ✅ เสร็จสิ้น | พนักงานทำเสร็จแล้ว | ตรวจสอบความถูกต้อง |
| ❌ ยกเลิก | งานถูกยกเลิก | - |

## 4. ดูรายงานปัญหา

### 4.1 หน้ารายงานปัญหา

ไปที่ **"รายงานปัญหา"** เพื่อดูปัญหาทั้งหมดที่ถูกแจ้งมา

### 4.2 ประเภทปัญหา

| ไอคอน | ประเภท | ตัวอย่าง |
|-------|--------|----------|
| 🦠 | โรค | โรคใบไหม้, โรครากเน่า |
| 🐛 | แมลง | หนอน, เพลี้ย, ไร |
| 💀 | ต้นไม้ตาย | ใบเหี่ยว, รากเน่า |
| 💧 | ปัญหาน้ำ | น้ำขัง, ดินแห้ง |
| 🧪 | ปัญหาปุ๋ย | ปุ๋ยไหม้, ขาดธาตุอาหาร |
| ⚠️ | อื่นๆ | ปัญหาอื่นที่ไม่อยู่ในประเภท |

### 4.3 ดำเนินการกับปัญหา

เมื่อคลิก **"ดำเนินการ"** คุณสามารถ:

| การดำเนินการ | คำอธิบาย |
|--------------|----------|
| **รับทราบ** | บอกว่าเห็นปัญหาแล้ว |
| **กำลังแก้ไข** | อยู่ระหว่างดำเนินการ |
| **มอบหมายให้คนอื่น** | ส่งต่อให้พนักงานอื่นดูแล |
| **ขอข้อมูลเพิ่มเติม** | ถามผู้แจ้งเพิ่มเติม |
| **ปิดปัญหา** | ปัญหาได้รับการแก้ไขแล้ว |

### 4.4 ระดับความรุนแรง

| ระดับ | สี | การดำเนินการ |
|-------|-----|--------------|
| **วิกฤต** | 🔴 | ดำเนินการทันที - โทรหาผู้แจ้ง |
| **สูง** | 🟠 | ดำเนินการภายใน 24 ชม. |
| **ปานกลาง** | 🟡 | ดำเนินการภายใน 2-3 วัน |
| **ต่ำ** | 🟢 | บันทึกไว้ตรวจสอบ |

## 5. ดู Log กิจกรรม

### 5.1 หน้า Log กิจกรรม

ไปที่ **"กิจกรรม" → "Log กิจกรรม"** เพื่อดูประวัติกิจกรรมทั้งหมด

### 5.2 ประเภทกิจกรรมที่บันทึก

| ไอคอน | กิจกรรม | ข้อมูลที่บันทึก |
|-------|--------|----------------|
| 💧 | รดน้ำ | ปริมาณน้ำ, ความชื้นดิน |
| 🧪 | ใส่ปุ๋ย | ชนิดปุ๋ย, ปริมาณ |
| ✂️ | ตัดแต่ง | ส่วนที่ตัด, เหตุผล |
| 🔬 | ตรวจสอบ | สภาพทั่วไป |
| 🐛 | กำจัดแมลง | ชนิดแมลง, วิธีการ |
| 💊 | พ่นยา | ชนิดยา, ปริมาณ |
| ⚠️ | แจ้งปัญหา | ประเภทปัญหา, ความรุนแรง |

### 5.3 กรอง Log กิจกรรม

| ตัวกรอง | ความหมาย |
|---------|----------|
| **พนักงาน** | ดูเฉพาะกิจกรรมของพนักงานคนใดคนหนึ่ง |
| **ประเภทกิจกรรม** | กรองเฉพาะ รดน้ำ/ใส่ปุ๋ย/ตัดแต่ง/อื่นๆ |
| **แปลง/พืช** | ดูเฉพาะกิจกรรมของแปลงหรือพืชใด |
| **ช่วงเวลา** | วันนี้, สัปดาห์นี้, เดือนนี้, กำหนดเอง |

### 5.4 Export ข้อมูล

หากต้องการ Export ข้อมูล:

1. คลิกปุ่ม **"📥 Export"**
2. เลือกรูปแบบไฟล์:
   - **CSV** - ใช้กับ Excel, Google Sheets
   - **Excel (.xlsx)** - สำหรับ Excel
   - **PDF** - สำหรับพิมพ์เอกสาร
3. เลือกช่วงวันที่
4. คลิก **"ดาวน์โหลด"**

## 6. จัดการ Plots และ Plants

### 6.1 จัดการแปลง (Plots)

ไปที่ **"แปลงและพืช" → "จัดการแปลง"**

**เพิ่มแปลงใหม่:**

1. คลิก **"+ เพิ่มแปลง"**
2. กรอกข้อมูล:

| ช่อง | คำอธิบาย |
|------|----------|
| **ชื่อแปลง** | ชื่อที่สื่อความหมาย |
| **ขนาด** | กว้าง × ยาว (เมตร) |
| **ประเภทพืชหลัก** | มะม่วง, ลำไย, ทุเรียน, อื่นๆ |
| **ที่ตั้งในฟาร์ม** | ด้านหน้า/หลัง/ข้าง |
| **หมายเหตุ** | ข้อมูลเพิ่มเติม |

**แก้ไข/ลบแปลง:**

1. คลิกที่แปลงที่ต้องการแก้ไข
2. คลิก **"แก้ไข"** หรือ **"ลบ"**
3. ยืนยันการดำเนินการ

> ⚠️ **ระวัง:** การลบแปลงจะลบข้อมูลพืชและประวัติทั้งหมดในแปลงนั้นด้วย

### 6.2 จัดการพืช (Plants)

ไปที่ **"แปลงและพืช" → "จัดการพืช"**

**เพิ่มพืชใหม่:**

1. เลือกแปลงที่ต้องการเพิ่มพืช
2. คลิก **"+ เพิ่มพืช"**
3. กรอกรายละเอียด:

| ช่อง | คำอธิบาย |
|------|----------|
| **ชนิดพันธุ์** | ชื่อพันธุ์พืช |
| **วันที่ปลูก** | วันที่เริ่มปลูก |
| **ตำแหน่งในแปลง** | แถว, คอลัมน์ |
| **สถานะ** | สุขภาพดี, เฝ้าระวัง, ป่วย, ตาย |

### 6.3 พิมพ์ QR Code

เมื่อเพิ่มพืชแล้ว ระบบจะสร้าง QR Code ให้อัตโนมัติ:

1. เลือกพืชที่ต้องการพิมพ์ QR Code
2. คลิก **"🖨️ พิมพ์ QR Code"**
3. เลือกจำนวน:
   - **พิมพ์ทีละต้น** - พิมพ์ QR Code 1 ต้น
   - **พิมพ์ทั้งแปลง** - พิมพ์ QR Code ทั้งแปลง
4. คลิก **"พิมพ์"** หรือ **"ดาวน์โหลด PDF"**

> 💡 **เคล็ดลับ:** ควรพิมพ์ QR Code ลงบนกระดาษชนิดทนน้ำ

## 7. ดู Yield Reports

### 7.1 หน้า Yield Report

ไปที่ **"รายงาน" → "Yield Report"** เพื่อดูรายงานผลผลิต

### 7.2 ข้อมูลที่แสดงใน Yield Report

| ข้อมูล | คำอธิบาย |
|--------|----------|
| **ผลผลิตรวม** | ปริมาณผลผลิตทั้งหมด (กก./ตัน) |
| **ผลผลิตต่อต้น** | ค่าเฉลี่ยผลผลิตต่อต้น |
| **ผลผลิตต่อแปลง** | เปรียบเทียบผลผลิตระหว่างแปลง |
| **แนวโน้ม** | กราฟแสดงแนวโน้มผลผลิตรายเดือน/รายปี |
| **เปรียบเทียบพันธุ์** | เปรียบเทียบผลผลิตระหว่างพันธุ์ |

### 7.3 ตัวกรองข้อมูล

| ตัวกรอง | ความหมาย |
|---------|----------|
| **ช่วงเวลา** | รายเดือน, รายไตรมาส, รายปี |
| **แปลง** | เลือกเฉพาะแปลงที่สนใจ |
| **พันธุ์** | เลือกเฉพาะพันธุ์ที่สนใจ |

### 7.4 Export ข้อมูล

1. คลิกปุ่ม **"📥 Export"**
2. เลือกรูปแบบไฟล์: CSV, Excel, PDF
3. คลิก **"ดาวน์โหลด"**

## ❓ คำถามที่พบบ่อย

**Q: พนักงานไม่เห็นงานที่มอบหมาย ต้องทำอย่างไร?**
> A: ตรวจสอบว่าได้เลือกพนักงานถูกต้อง และตรวจสอบว่าพนักงานล็อกอินด้วยบัญชีที่ถูกต้อง

**Q: สามารถเพิ่มพืชหลายต้นพร้อมกันได้ไหม?**
> A: ได้ ใช้ฟังก์ชัน "นำเข้าจาก Excel" หรือ "Bulk Add"

**Q: หากต้องการเปลี่ยนพนักงานรับผิดชอบแปลง ต้องทำอย่างไร?**
> A: ไปที่จัดการแปลง → เลือกแปลง → แก้ไขช่อง "ผู้ดูแล" หรือ "ผู้รับผิดชอบ"
