## 1. Dashboard - ภาพรวมทั้งระบบ

### 1.1 หน้า Dashboard หลัก

เมื่อเข้าสู่ระบบในฐานะ Owner คุณจะเห็นภาพรวมทั้งระบบทั้งหมด

### 1.2 วิดเจ็ต Dashboard สำหรับ Owner

| วิดเจ็ต | ข้อมูลที่แสดง |
|--------|--------------|
| **Stats Cards** | จำนวนฟาร์ม, พนักงาน, งาน, ต้นไม้ (รวมทั้งระบบ) |
| **สถานะงาน** | % งานเสร็จ/กำลังทำ/รอดำเนิน |
| **สถานะพืช** | % สุขภาพดี/เฝ้าระวัง/มีปัญหา |
| **ภาพรวมแต่ละฟาร์ม** | สถานะโดยย่อของแต่ละฟาร์ม |
| **รายงานปัญหา** | ปัญหาล่าสุดจากทุกฟาร์ม |

### 1.3 ฟีเจอร์พิเศษสำหรับ Owner

- **ดูข้อมูลข้ามฟาร์ม:** เปรียบเทียบประสิทธิภาพระหว่างฟาร์ม
- **รายงานรวม:** ดูรายงานทั้งหมดในหน้าเดียว
- **จัดการผู้ใช้ระดับสูง:** เพิ่ม/ลบ Manager และ Owner

## 2. จัดการ Users (เพิ่ม/ลบ/เปลี่ยน Role)

### 2.1 หน้าจัดการผู้ใช้

ไปที่ **"ตั้งค่า" → "จัดการผู้ใช้"**

### 2.2 บทบาท (Role) ในระบบ

| Role | สิทธิ์การใช้งาน |
|------|----------------|
| **👑 Owner** | ทุกสิทธิ์ - จัดการระบบทั้งหมด, ผู้ใช้, ฟาร์ม |
| **🏢 Manager** | จัดการงาน, พืช, แปลง, ดูรายงาน (เฉพาะฟาร์มที่ได้รับมอบหมาย) |
| **👷 Worker** | ดูงาน, บันทึกกิจกรรม, แจ้งปัญหา (เฉพาะฟาร์มที่ได้รับมอบหมาย) |

### 2.3 เพิ่มผู้ใช้ใหม่

**ขั้นตอนที่ 1:** คลิก **"+ เพิ่มผู้ใช้ใหม่"**

**ขั้นตอนที่ 2:** กรอกรายละเอียดผู้ใช้

| ช่อง | คำอธิบาย |
|------|----------|
| **ชื่อ-นามสกุล** | ชื่อผู้ใช้ |
| **อีเมล** | อีเมลสำหรับติดต่อ |
| **เบอร์โทร** | เบอร์โทรศัพท์ |
| **Username** | ชื่อผู้ใช้สำหรับเข้าสู่ระบบ |
| **Password** | รหัสผ่าน (อย่างน้อย 8 ตัวอักษร) |
| **บทบาท** | เลือก Owner/Manager/Worker |
| **ฟาร์มที่เข้าถึงได้** | เลือกฟาร์มที่ผู้ใช้เข้าถึงได้ |

**ขั้นตอนที่ 3:** คลิก **"สร้างผู้ใช้"**

> ✅ ระบบจะส่งอีเมลพร้อมข้อมูลการเข้าใช้งานให้ผู้ใช้ใหม่โดยอัตโนมัติ

### 2.4 แก้ไขผู้ใช้

1. ไปที่ **"ตั้งค่า" → "จัดการผู้ใช้"**
2. คลิก **"✏️ แก้ไข"** ที่ผู้ใช้ที่ต้องการ
3. แก้ไขข้อมูลที่ต้องการ:

| ช่องที่แก้ไขได้ | คำอธิบาย |
|----------------|----------|
| **ชื่อ-นามสกุล** | ชื่อผู้ใช้ |
| **อีเมล** | อีเมลสำหรับติดต่อ |
| **เบอร์โทร** | เบอร์โทรศัพท์ |
| **บทบาท** | เปลี่ยน Role (เช่น Worker → Manager) |
| **ฟาร์มที่เข้าถึง** | กำหนดว่าผู้ใช้เข้าถึงฟาร์มไหนได้บ้าง |
| **สถานะ** | เปิด/ปิดการใช้งาน |

4. คลิก **"บันทึก"**

### 2.5 เปลี่ยนรหัสผ่านผู้ใช้

1. คลิก **"✏️ แก้ไข"** ที่ผู้ใช้
2. คลิก **"🔐 เปลี่ยนรหัสผ่าน"**
3. กรอกรหัสผ่านใหม่ 2 ครั้ง
4. คลิก **"บันทึก"**

> 💡 **เคล็ดลับ:** หากผู้ใช้ลืมรหัสผ่าน สามารถกด **"ส่งลิงก์ตั้งรหัสผ่านใหม่"** เพื่อส่งอีเมลตั้งรหัสผ่านให้

### 2.6 ลบผู้ใช้

1. ไปที่ **"ตั้งค่า" → "จัดการผู้ใช้"**
2. คลิก **"🗑️ ลบ"** ที่ผู้ใช้ที่ต้องการลบ
3. เลือกวิธีจัดการงานของผู้ใช้คนนั้น:
   - **โอนงานให้คนอื่น** - เลือกพนักงานที่จะรับงานแทน
   - **ลบงานที่ยังไม่เสร็จ** - ลบงานที่ยังค้างอยู่ทิ้ง
4. คลิก **"ยืนยันลบ"**

> ⚠️ **หมายเหตุ:** ไม่สามารถลบ Owner คนสุดท้ายได้ ต้องมี Owner อย่างน้อย 1 คนในระบบ

## 3. จัดการ Farms

### 3.1 หน้าจัดการฟาร์ม

ไปที่ **"ตั้งค่า" → "จัดการฟาร์ม"**

### 3.2 เพิ่มฟาร์มใหม่

**ขั้นตอนที่ 1:** คลิก **"+ เพิ่มฟาร์มใหม่"**

**ขั้นตอนที่ 2:** กรอกรายละเอียดฟาร์ม

| ช่อง | คำอธิบาย |
|------|----------|
| **ชื่อฟาร์ม** | ชื่อที่สื่อความหมาย |
| **ที่อยู่** | ที่อยู่ฟาร์ม |
| **จังหวัด** | จังหวัดที่ตั้ง |
| **พื้นที่** | ขนาดพื้นที่ (ไร่) |
| **ละติจูด/ลองจิจูด** | พิกัดสำหรับแผนที่ |
| **ผู้จัดการฟาร์ม** | เลือก Manager ที่ดูแลฟาร์มนี้ |
| **โทรศัพท์/อีเมล** | ข้อมูลติดต่อฟาร์ม |

**ขั้นตอนที่ 3:** คลิก **"สร้างฟาร์ม"**

### 3.3 แก้ไขข้อมูลฟาร์ม

1. ไปที่ **"ตั้งค่า" → "จัดการฟาร์ม"**
2. คลิก **"✏️ แก้ไข"** ที่ฟาร์มที่ต้องการ
3. แก้ไขข้อมูลที่ต้องการ
4. คลิก **"บันทึก"**

### 3.4 ดูรายละเอียดฟาร์ม

คลิก **"👀 ดูรายละเอียด"** เพื่อดู:

| ข้อมูล | รายละเอียด |
|--------|------------|
| **ภาพรวม** | จำนวนแปลง, พืช, พนักงาน |
| **สถานะงาน** | งานทั้งหมดในฟาร์ม |
| **รายงานปัญหา** | ปัญหาทั้งหมดในฟาร์ม |
| **กิจกรรมล่าสุด** | Log กิจกรรมในฟาร์ม |
| **รายงานผลผลิต** | Yield Report ของฟาร์ม |

### 3.5 ลบฟาร์ม

> ⚠️ **คำเตือน:** การลบฟาร์มจะลบข้อมูลทั้งหมดรวมถึงแปลง, พืช, งาน และประวัติกิจกรรม

1. ไปที่ **"ตั้งค่า → จัดการฟาร์ม"**
2. คลิก **"🗑️ ลบ"** ที่ฟาร์มที่ต้องการลบ
3. ระบบจะถามยืนยัน
4. คลิก **"ยืนยันลบ"**

## 4. ดู Reports ทั้งหมด

### 4.1 หน้ารายงาน

ไปที่ **"รายงาน"** เพื่อดูรายงานทั้งหมดในระบบ

### 4.2 ประเภทรายงาน

| รายงาน | คำอธิบาย |
|--------|----------|
| **📊 สรุปภาพรวม** | สถานะทั้งระบบ ณ ปัจจุบัน |
| **📋 รายงานงาน** | สถานะและความคืบหน้างานทั้งหมด |
| **🌱 รายงานพืช** | สถานะสุขภาพพืชทั้งหมด |
| **⚠️ รายงานปัญหา** | สรุปปัญหาทั้งหมดที่แจ้งมา |
| **📈 Yield Report** | รายงานผลผลิต |
| **👥 รายงานพนักงาน** | ประสิทธิภาพการทำงานของพนักงาน |
| **💧 รายงานการใช้น้ำ** | สรุปการใช้น้ำในฟาร์ม |
| **🧪 รายงานการใช้ปุ๋ย/ยา** | สรุปการใช้ปุ๋ยและยาฆ่าแมลง |

### 4.3 รายงานสรุปภาพรวม (Executive Summary)

| ข้อมูล | รายละเอียด |
|--------|------------|
| **ผลผลิตรวม** | ปริมาณผลผลิตทั้งหมด (กก./ตัน) + % เปลี่ยนแปลงจากเดือนก่อน |
| **ประสิทธิภาพ** | % งานเสร็จ, % ปัญหาที่แก้ไขได้ |
| **สถานะพืช** | จำนวนต้นแยกตามสถานะ (ดี/เฝ้าระวัง/ป่วย) แยกตามชนิดพืช |
| **ปัญหาที่ต้องติดตาม** | ประเภทปัญหา + % การเปลี่ยนแปลง |

### 4.4 รายงานพนักงาน (Performance Report)

| เมตริก | คำอธิบาย |
|--------|----------|
| **จำนวนงานที่ทำเสร็จ** | งานที่พนักงานทำเสร็จในช่วงเวลาที่เลือก |
| **เวลาเฉลี่ยต่องาน** | ค่าเฉลี่ยเวลาที่ใช้ต่องาน |
| **คุณภาพงาน** | คะแนนความถูกต้องของงาน |
| **จำนวนปัญหาที่แจ้ง** | ปัญหาที่พนักงานพบและแจ้ง |
| **อัตราการลาออก** | สถิติการลาออก/ลาพัก |

### 4.5 Export รายงาน

1. ไปที่หน้ารายงานที่ต้องการ
2. เลือกช่วงเวลา/ฟาร์มที่ต้องการ
3. คลิก **"📥 Export"**
4. เลือกรูปแบบไฟล์:
   - **CSV** - สำหรับ Excel, Google Sheets
   - **Excel (.xlsx)** - สำหรับ Excel
   - **PDF** - สำหรับพิมพ์เอกสาร
5. คลิก **"ดาวน์โหลด"**

## 5. ตั้งค่าระบบ

### 5.1 หน้าตั้งค่าระบบ

ไปที่ **"ตั้งค่า" → "ตั้งค่าระบบ"**

### 5.2 ตั้งค่าทั่วไป

| ช่องตั้งค่า | คำอธิบาย |
|-------------|----------|
| **ชื่อระบบ** | ชื่อที่แสดงบนแถบเบราว์เซอร์ |
| **โลโก้** | รูปโลโก้ขององค์กร |
| **โซนเวลา** | เขตเวลาของระบบ |
| **รูปแบบวันที่** | วัน/เดือน/ปี หรือ เดือน/วัน/ปี |
| **ภาษา** | ภาษาที่ใช้ในระบบ (ไทย/อังกฤษ) |

### 5.3 ตั้งค่าการแจ้งเตือน

| ช่องตั้งค่า | คำอธิบาย |
|-------------|----------|
| **อีเมลแจ้งเตือน** | ส่งอีเมลเมื่อมีปัญหาใหม่ |
| **LINE Notify** | เชื่อมต่อ LINE Notify สำหรับแจ้งเตือน |
| **แจ้งเตือนภายในระบบ** | แจ้งเตือนในหน้าเว็บ |
| **เสียงแจ้งเตือน** | เปิด/ปิดเสียงเมื่อมีแจ้งเตือนใหม่ |

### 5.4 ตั้งค่าการยืนยันตัวตน (Authentication)

| ช่องตั้งค่า | คำอธิบาย |
|-------------|----------|
| **อนุญาตให้ล็อกอินผ่าน LINE** | ให้ผู้ใช้ล็อกอินด้วย LINE |
| **อนุญาตให้ล็อกอินด้วย Google** | ให้ผู้ใช้ล็อกอินด้วย Google |
| **บังคับเปลี่ยนรหัสทุก 90 วัน** | บังคับให้เปลี่ยนรหัสผ่านเป็นประจำ |
| **ล็อกอินหลังใส่รหัสผิด 5 ครั้ง** | ป้องกันการเจาะระบบ |

### 5.5 ตั้งค่าการจัดการงาน

| ช่องตั้งค่า | คำอธิบาย |
|-------------|----------|
| **แจ้งเตือนก่อนครบกำหนด** | แจ้งเตือนก่อนงานครบกำหนด (ชั่วโมง) |
| **ส่งอีเมลเมื่องานถูกมอบหมาย** | แจ้งพนักงานเมื่อได้รับงานใหม่ |
| **อนุญาต Worker สร้างงานเอง** | ให้ Worker สร้างงานใหม่ได้ |
| **ต้องอนุมัติก่อนปิดงาน** | Manager ต้องอนุมัติก่อนงานเสร็จ |

### 5.6 ตั้งค่าการรายงานปัญหา

| ช่องตั้งค่า | คำอธิบาย |
|-------------|----------|
| **ระดับความรุนแรงเริ่มต้น** | ความรุนแรงเริ่มต้นเมื่อแจ้งปัญหา |
| **แจ้ง Owner เมื่อปัญหาระดับวิกฤต** | ส่งอีเมล/LINE ให้ Owner เมื่อปัญหาวิกฤต |
| **กำหนดเวลาแก้ไขปัญหาตามระดับ** | กำหนดเวลาตามความรุนแรง (ชั่วโมง) |

### 5.7 ตั้งค่าการสำรองข้อมูล (Backup)

| ช่องตั้งค่า | คำอธิบาย |
|-------------|----------|
| **Backup อัตโนมัติ** | ทำ backup อัตโนมัติทุกวัน |
| **เก็บ Backup ย้อนหลัง** | จำนวนวันที่เก็บ Backup |
| **ส่ง Backup ไป Cloud** | ส่งไป Google Drive/Dropbox |
| **อีเมลแจ้งเตือน Backup ล้มเหลว** | แจ้งเตือนเมื่อ backup ไม่สำเร็จ |

### 5.8 บันทึกการตั้งค่า

หลังจากแก้ไขการตั้งค่าใดๆ:

1. คลิก **"บันทึก"** ที่ด้านล่างของหน้า
2. ระบบจะแจ้ง "บันทึกสำเร็จ"
3. การตั้งค่าจะมีผลทันที

> 💡 **เคล็ดลับ:** ควรทำการตั้งค่าทีละส่วน และกดบันทึกหลังแก้ไขแต่ละส่วน เพื่อป้องกันการสูญหายของการตั้งค่า

## 6. การบริหารระบบขั้นสูง

### 6.1 ดู Log การใช้งาน (Audit Log)

ไปที่ **"ตั้งค่า" → "Audit Log"** เพื่อดูประวัติการดำเนินการต่างๆ ในระบบ

| ข้อมูลที่บันทึก | รายละเอียด |
|----------------|------------|
| **ใคร** | ชื่อผู้ใช้ที่ทำการ |
| **ทำอะไร** | การดำเนินการ (เพิ่ม/แก้ไข/ลบ) |
| **อะไร** | รายการที่ถูกดำเนินการ (งาน/ผู้ใช้/ฟาร์ม) |
| **เมื่อไหร่** | วันเวลาที่ทำการ |
| **จากไหน** | IP Address / อุปกรณ์ |

### 6.2 ดูสถิติการใช้งาน

ไปที่ **"ตั้งค่า" → "สถิติระบบ"** เพื่อดู:

- จำนวนผู้ใช้ที่เข้าใช้งานรายวัน/รายสัปดาห์/รายเดือน
- ค่าเฉลี่ยเวลาที่ใช้ในระบบ
- ฟีเจอร์ที่ใช้บ่อยที่สุด
- อุปกรณ์ที่ใช้เข้าสู่ระบบ

## ❓ คำถามที่พบบ่อย

**Q: สามารถมี Owner หลายคนได้ไหม?**
> A: ได้ ระบบรองรับ Owner หลายคน ทุกคนมีสิทธิ์เท่ากัน

**Q: หาก Manager ลาออก ต้องทำอย่างไร?**
> A: 1) ลบบัญชี Manager 2) โอนงานให้ Manager คนใหม่ หรือ Owner ดูแลชั่วคราว

**Q: สามารถตั้งค่าให้ Worker ไม่เห็นข้อมูลฟาร์มอื่นได้ไหม?**
> A: ได้ กำหนดได้ที่ "ฟาร์มที่เข้าถึงได้" ในการตั้งค่าผู้ใช้

**Q: Backup ข้อมูลถูกเก็บที่ไหน?**
> A: สามารถตั้งค่าให้เก็บที่ Server หรือ Cloud (Google Drive/Dropbox)
