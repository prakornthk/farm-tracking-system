# 📖 คู่มือเริ่มต้นใช้งานด่วน - Farm Tracking System

## ระบบ Farm Tracking System คืออะไร?

Farm Tracking System เป็นระบบจัดการฟาร์มแบบครบวงจร ช่วยให้คุณสามารถ:

- ✅ ติดตามและมอบหมายงานในฟาร์ม
- ✅ บันทึกกิจกรรมการดูแลพืช (รดน้ำ, ใส่ปุ๋ย, ตัดแต่ง)
- ✅ รายงานปัญหาที่พบ (โรค, แมลง, ต้นไม้ตาย)
- ✅ ดูภาพรวมสถานะฟาร์มแบบเรียลไทม์
- ✅ วิเคราะห์ผลผลิตและประสิทธิภาพการทำงาน

---

## 🚀 วิธีเข้าใช้งานครั้งแรก

### ขั้นตอนที่ 1: เข้าสู่ระบบ

**สำหรับ Web Admin:**
1. เปิดเบราว์เซอร์ไปที่ `http://admin.farm.localhost`
2. กรอก **Username** และ **Password** ที่ได้รับจากผู้ดูแลระบบ
3. คลิกปุ่ม **"เข้าสู่ระบบ"**

**สำหรับ LINE App (LIFF):**
1. เปิด LINE และไปที่ **@Farm Tracking** (หรือลิงก์ที่ได้รับ)
2. ระบบจะเปิด LIFF App ใน LINE
3. คลิก **"ล็อกอิน"** เพื่อเข้าใช้งาน

### ขั้นตอนที่ 2: เปลี่ยนรหัสผ่าน (แนะนำ)

> ⚠️ หากเป็นการเข้าใช้ครั้งแรก ควรเปลี่ยนรหัสผ่านทันที

1. คลิกที่ **ไอคอนโปรไฟล์** มุมบนขวา
2. เลือก **"ตั้งค่าบัญชี"**
3. กรอกรหัสผ่านใหม่และยืนยัน
4. คลิก **"บันทึก"**

## 🔄 Quick Tour ของระบบ

### เมนูหลัก

```
📋 งาน (Tasks)
   ├── 📋 รายการงาน - ดูและจัดการงานทั้งหมด
   ├── ➕ สร้างงานใหม่ - มอบหมายงานให้พนักงาน
   └── 📊 สถานะงาน - ติดตามความคืบหน้า

🌱 แปลงและพืช (Plots & Plants)
   ├── 🗺️ แผนที่ฟาร์ม - ดูผังการจัดวางแปลง
   ├── 🌿 รายการพืช - ดูพืชทั้งหมดในฟาร์ม
   └── 📈 Yield Report - ดูรายงานผลผลิต

⚠️ รายงานปัญหา (Issues)
   ├── 📋 รายการปัญหา - ดูปัญหาทั้งหมด
   └── ➕ แจ้งปัญหาใหม่ - รายงานปัญหาพบเจอ

👥 ผู้ใช้ (Users) - สำหรับ Manager/Owner
   ├── 📋 รายชื่อผู้ใช้ - ดูและจัดการผู้ใช้
   └── ⚙️ ตั้งค่า - จัดการบทบาทและสิทธิ์

⚙️ ตั้งค่า (Settings)
   ├── 🏠 ข้อมูลฟาร์ม - จัดการข้อมูลฟาร์ม
   ├── 🔔 การแจ้งเตือน - ตั้งค่าการแจ้งเตือน
   └── 🔐 บัญชีของฉัน - จัดการข้อมูลส่วนตัว
```

### สำหรับแต่ละบทบาท (Role)

| บทบาท | เมนูที่เข้าถึงได้ |
|-------|-----------------|
| **Worker** | งานของฉัน, บันทึกกิจกรรม, สแกน QR, แจ้งปัญหา |
| **Manager** | Worker + จัดการงาน, ดูรายงาน, จัดการแปลง/พืช |
| **Owner** | ทุกเมนู + จัดการผู้ใช้, จัดการฟาร์ม, ตั้งค่าระบบ |

## 🔑 ข้อมูลสำคัญ

### การใช้ QR Code

ทุกแปลงและพืชมี **QR Code** ติดอยู่ สามารถสแกนเพื่อ:

- ดูข้อมูลพืช/แปลง
- บันทึกกิจกรรมได้อย่างรวดเร็ว
- รายงานปัญหา ณ จุดที่พบ

### สัญลักษณ์สถานะ

| สัญลักษณ์ | สถานะ | ความหมาย |
|-----------|-------|----------|
| 🔴 | วิกฤต | ต้องดำเนินการทันที |
| 🟡 | เตือน | ต้องจัดการเร็วๆ นี้ |
| 🟢 | ปกติ | ไม่มีปัญหา |
| 🔵 | กำลังดำเนินการ | อยู่ระหว่างทำงาน |
