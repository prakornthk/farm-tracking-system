## 1. LIFF App คืออะไร

**LIFF (LINE Front-end Framework)** คือ แพลตฟอร์มสำหรับสร้าง web app ที่ทำงานภายใน LINE ซึ่งช่วยให้ผู้ใช้งานสามารถเข้าถึงฟังก์ชันต่าง ๆ ของระบบจัดการฟาร์มได้โดยตรงจากแอป LINE โดยไม่ต้องติดตั้งแอปเพิ่มเติม

### ข้อดีของ LIFF App

- ไม่ต้องติดตั้งเพิ่ม - ทำงานใน LINE ที่มีอยู่แล้ว
- เข้าถึงง่าย - เปิดผ่าน QR Code หรือ Deep Link
- เชื่อมกับ LINE Account - ล็อกอินด้วย LINE ID อัตโนมัติ
- รองรับ Offline - บันทึกข้อมูลได้แม้ไม่มีเน็ต
- ใช้งานบนมือถือ - ออกแบบมาสำหรับหน้าจอเล็ก

### ฟังก์ชันหลักที่รองรับ

- บันทึกกิจกรรมการเกษตร
- รายงานปัญหา/ความผิดปกติ
- ดูงานที่ได้รับมอบหมาย
- ดูสถานะแปลงปลูกและพืช
- สแกน QR Code บนแปลง
- ทำงาน Offline ได้

## 2. วิธีเปิดใช้งาน LIFF

### ข้อกำหนดเบื้องต้น

- แอป LINE: เวอร์ชันล่าสุด
- ระบบปฏิบัติการ: iOS 12.0+ หรือ Android 8.0+
- อินเทอร์เน็ต: จำเป็นสำหรับการเปิดใช้งานครั้งแรก

### วิธีที่ 1: เปิดผ่าน QR Code (แนะนำ)

1. เปิดแอป LINE บนมือถือของคุณ
2. แตะไอคอน + (เพิ่มเพื่อน) ที่ด้านบน
3. เลือก "สแกน QR Code"
4. สแกน QR Code ที่ได้รับจากผู้ดูแลระบบ
5. แตะ "เพิ่ม" เพื่อเปิด LIFF App
6. ระบบจะเปิด LIFF App อัตโนมัติ

### วิธีที่ 2: เปิดผ่าน Deep Link

1. คลิกลิงก์ ที่ได้รับใน LINE Chat
2. ยืนยันการเปิด ใน LINE
3. รอสักครู่ ให้ LIFF App โหลด

### วิธีที่ 3: ค้นหาจาก Chat List

1. เปิด LINE แล้วไปที่ Chat
2. ค้นหาชื่อ LIFF App ในกล่องค้นหา
3. แตะเปิด LIFF App

### กรณีเปิดไม่ได้

- ตรวจสอบว่าเชื่อมต่ออินเทอร์เน็ตแล้ว
- อัปเดต LINE เป็นเวอร์ชันล่าสุด
- ล้าง cache ของ LINE (ตั้งค่า > แอป > LINE > ล้าง cache)

## 3. วิธีสแกน QR Code

### QR Code ในระบบ

ระบบจัดการฟาร์มมี QR Code สำหรับระบุ:

- Farm QR - ระบุตำแหน่งฟาร์ม
- Zone QR - ระบุโซนภายในฟาร์ม
- Plot QR - ระบุแปลงปลูก
- Plant QR - ระบุต้นไม้/พืชเฉพาะต้น
- Activity QR - บันทึกกิจกรรมในจุดนั้น ๆ

### วิธีสแกนใน LIFF App

1. เปิด LIFF App จาก LINE
2. แตะปุ่ม "สแกน" ที่หน้าหลัก หรือ Quick Action
3. อนุญาตการเข้าถึงกล้อง (ครั้งแรกเท่านั้น)
4. สแกน QR Code โดยเล็งให้ QR อยู่ในกรอบ
5. ระบบจะแสดงข้อมูล ของจุดที่สแกน

### หลังสแกน

```
+----------------------------+
| สแกนสำเร็จ!                |
+----------------------------+
| ฟาร์ม: สวนผลไม้ลุงสมชาย |
| โซน: โซน A - สวนมะม่วง  |
| แปลง: แปลง A-01           |
| พืช: มะม่วงนวล (ต้นที่ 15)|
+----------------------------+
| [บันทึกกิจกรรม][รายงานปัญหา]|
+----------------------------+
```

### เคล็ดลับการสแกน

- แสงสว่าง - สแกนในที่ที่มีแสงเพียงพอ
- ระยะห่าง - ถือกล้องห่าง QR Code ประมาณ 20-30 ซม.
- โทรศัพท์ตั้งขึ้น - ถือโทรศัพท์ตรง ๆ อย่าคว่ำ
- ถ้าสแกนไม่ได้ - ลองปรับความสว่างหน้าจอ

## 4. Quick Actions - บันทึกกิจกรรม

### Quick Actions คืออะไร

Quick Actions เป็นปุ่มลัดสำหรับบันทึกกิจกรรมได้อย่างรวดเร็ว โดยไม่ต้องผ่านหลายขั้นตอน

### วิธีใช้ Quick Actions

1. เปิด LIFF App หน้าหลัก
2. แตะปุ่ม Quick Actions (ไอคอน ⚡ หรือ ⚙️)
3. เลือกประเภทกิจกรรม

### ประเภทกิจกรรมที่บันทึกได้

| ไอคอน | กิจกรรม | รายละเอียด |
|-------|--------|------------|
| 💧 | รดน้ำ | บันทึกการรดน้ำพืช |
| 🧪 | ใส่ปุ๋ย | บันทึกการใส่ปุ๋ย |
| ✂️ | ตัดแต่ง | บันทึกการตัดแต่งกิ่ง |
| 🐛 | พ่นยา | บันทึกการพ่นสารเคมี/ยา |
| 🔍 | ตรวจสอบ | ตรวจสอบสถานะพืช |
| 📝 | อื่น ๆ | บันทึกกิจกรรมอื่น ๆ |

### ขั้นตอนบันทึกกิจกรรม

1. เลือกประเภทกิจกรรม (เช่น รดน้ำ 💧)
2. เลือกพื้นที่/แปลง (จากรายการ หรือ สแกน QR)
3. ระบุรายละเอียด (ปริมาณน้ำ, ชนิดปุ๋ย, หมายเหตุ)
4. ถ่ายรูป (ถ้ามี) แนบรูปถ่ายสถานที่จริง
5. ยืนยัน แตะ "บันทึก"

### สิ่งที่จะเก็บอัตโนมัติ

เมื่อบันทึกสำเร็จ ระบบจะบันทึกข้อมูลต่อไปนี้:

- ผู้บันทึก - ชื่อ LINE ของคนที่บันทึก
- ตำแหน่ง - พิกัด GPS (ถ้าเปิด Location)
- วันเวลา - วันที่และเวลาที่บันทึก
- อุปกรณ์ - รุ่นโทรศัพท์ที่ใช้

## 5. รายงานปัญหา

### รายงานปัญหาคืออะไร

ระบบให้ผู้ใช้งานรายงานปัญหาหรือความผิดปกติที่พบในฟาร์ม เพื่อให้ผู้ดูแลระบบและเจ้าของฟาร์มติดตามและแก้ไขได้ทันท่วงที

### ประเภทปัญหาที่รายงานได้

| ไอคอน | ประเภทปัญหา | ตัวอย่าง |
|-------|------------|---------|
| 🐛 | โรค/แมลง | หนอนกินใบ, โรคราสนิม |
| 💧 | น้ำ | น้ำขัง, ระบบน้ำเสียหาย |
| 🌡 | สภาพอากาศ | หนาวจัด, ร้อนจัด, ลมแรง |
| 🌱 | พืช | ใบเหลือง, ต้นแห้ง, ผลไม้เสีย |
| 🔧 | อุปกรณ์ | สปริงเกอร์เสีย, ท่อรั่ว |
| ⚡ | อื่น ๆ | ปัญหาอื่น ๆ ที่ไม่เข้ากลุ่ม |

### วิธีรายงานปัญหา

1. แตะปุ่ม "รายงานปัญหา" ที่หน้าหลักหรือหลังสแกน QR
2. เลือกประเภทปัญหาที่ตรงกับปัญหามากที่สุด
3. ระบุตำแหน่ง (สแกน QR หรือเลือกจากรายการ หรือเปิด GPS)
4. อธิบายปัญหา ระบุรายละเอียดให้ชัดเจน
5. ถ่ายรูปปัญหา ให้เห็นปัญหาชัดเจน
6. ระบุระดับความเร่งด่วน:
   - ด่วนมาก - ส่งแจ้งเตือนทันที
   - ด่วน - ส่งแจ้งเตือนปกติ
   - ไม่ด่วน - บันทึกไว้ตรวจสอบ
7. ยืนยันการรายงาน แตะ "ส่งรายงาน"

### สถานะการรายงาน

| สถานะ | สี | ความหมาย |
|-------|-----|---------|
| รอตรวจสอบ | เทา | รอผู้ดูแลรับเรื่อง |
| กำลังดำเนินการ | เหลือง | ผู้ดูแลกำลังจัดการ |
| แก้ไขแล้ว | เขียว | ปัญหาได้รับการแก้ไขแล้ว |
| ไม่อนุมัติ | แดง | ไม่เป็นปัญหาจริง/ซ้ำ |

## 6. ดูงานที่ได้รับมอบหมาย

### งานที่ได้รับมอบหมายคืออะไร

เมื่อผู้ดูแลระบบหรือ Manager มอบหมายงานให้ผู้ใช้ LIFF App จะสามารถดูรายการงานที่ได้รับมอบหมายได้ในหน้านี้

### วิธีดูงานที่ได้รับมอบหมาย

1. เปิด LIFF App
2. แตะเมนู "งานของฉัน" หรือไอคอน 📋
3. ดูรายการงานที่ได้รับมอบหมาย

### วิธีทำงานให้เสร็จ

1. แตะ "เริ่มทำ" เพื่อเริ่มทำงาน
2. ทำงานตามที่ได้รับมอบหมาย
3. แตะ "เสร็จแล้ว" เมื่อทำเสร็จ
4. ถ่ายรูปหลักฐาน (ถ้ามี)
5. ระบุหมายเหตุ (ถ้ามี)

### การแจ้งเตือน

ระบบจะส่งการแจ้งเตือนใน LINE เมื่อ:

- มีงานใหม่มอบหมาย: ข้อความ + รูปงาน
- งานใกล้ถึงกำหนด: เตือนก่อน 1 ชั่วโมง
- งานเลยกำหนด: แจ้งเตือนด่วน
- งานถูกยกเลิก: ข้อความแจ้งยกเลิก

## 7. Offline Mode - ใช้งานเมื่อไม่มีเน็ต

### Offline Mode คืออะไร

Offline Mode คือฟีเจอร์ที่ช่วยให้ผู้ใช้งาน LIFF App สามารถบันทึกกิจกรรมและรายงานปัญหาได้แม้ไม่มีสัญญาณอินเทอร์เน็ต โดยข้อมูลจะถูกเก็บไว้ในเครื่องและ sync ขึ้นระบบเมื่อกลับมาออนไลน์

### ฟีเจอร์ที่ทำงาน Offline ได้

- บันทึกกิจกรรม: บันทึกได้ปกติ
- รายงานปัญหา: บันทึกได้ปกติ
- ดูงานที่ได้รับ: ดูได้เฉพาะที่โหลดไว้
- สแกน QR: ต้องออนไลน์
- ดู Dashboard: ต้องออนไลน์

### วิธีบันทึกกิจกรรม Offline

1. เปิด LIFF App (แม้ไม่มีเน็ต)
2. ระบบจะแสดง "โหมดออฟไลน์" ที่ด้านบน
3. ทำตามขั้นตอนปกติในการบันทึกกิจกรรม
4. เมื่อกดบันทึก จะมีข้อความบอกว่าบันทึกในเครื่องแล้ว และจะ sync เมื่อกลับมาออนไลน์

### ข้อมูลที่เก็บ Offline

เมื่อบันทึก Offline ระบบจะเก็บข้อมูลดังนี้:

- ตำแหน่ง: พิกัด GPS ณ จุดที่บันทึก
- รายละเอียด: ข้อมูลที่กรอก
- รูป: รูปที่ถ่าย (บีบอัดเล็กลง)
- เวลา: วันเวลาจริงที่บันทึก
- สถานะ: "รอ sync"

### ข้อจำกัดของ Offline Mode

- สแกน QR ไม่ได้: ต้องออนไลน์
- ดูข้อมูลล่าสุดไม่ได้: ดูได้เฉพาะที่โหลดไว้
- รูปถ่ายถูกบีบอัด: อาจมีคุณภาพลดลง
- ไม่รู้ว่ามีงานใหม่: ไม่มีการแจ้งเตือน
- ข้อมูลจะสูญถ้าเคลียร์ cache: ควร sync ก่อนเคลียร์

### เคล็ดลับการใช้ Offline Mode

1. เปิด LIFF App ก่อนเข้าพื้นที่ไม่มีเน็ต ให้โหลดข้อมูลล่าสุดไว้
2. ถ่ายรูปให้ชัด รูปจะถูกบีบอัด ถ่ายให้เห็นชัด
3. พก Power Bank ถ้าต้องทำงานนอกพื้นที่นาน ๆ
4. เช็คสถานะ Sync ก่อนกลับเข้าพื้นที่มีเน็ต

## 8. Sync เมื่อกลับมาออนไลน์

### Sync คืออะไร

Sync คือกระบวนการส่งข้อมูลที่บันทึกไว้ในโหมด Offline ขึ้นไปเก็บในระบบหลัก เมื่อกลับมาออนไลน์

### Sync อัตโนมัติ

เมื่อ LIFF App ตรวจพบว่ามีเน็ตแล้ว จะ sync ข้อมูลอัตโนมัติ:

```
+----------------------------------+
| กำลัง sync ข้อมูล...           |
|                                  |
| ส่งข้อมูลไประบบแล้ว            |
| [ดูรายการที่ sync แล้ว]         |
+----------------------------------+
```

### สถานะการ Sync

| สถานะ | ความหมาย |
|--------|---------|
| รอ sync | รอเชื่อมต่ออินเทอร์เน็ต |
| กำลัง sync | กำลันส่งข้อมูลไประบบ |
| sync สำเร็จ | ข้อมูลถูกส่งเรียบร้อยแล้ว |
| sync ล้มเหลว | เกิดข้อผิดพลาด กด retry |

### วิธี Sync ด้วยตนเอง

1. เปิด LIFF App
2. แตะแปรงสถานะ "รอ sync" ที่ด้านบน
3. แตะปุ่ม "Sync ตอนนี้"
4. รอให้ระบบส่งข้อมูลเสร็จ

### เคล็ดลับการ Sync

- ควร sync ในพื้นที่ที่มีสัญญาณเน็ตที่ดี
- หลีกเลี่ยงการปิดแอปขณะ sync
- ถ้า sync ล้มเหลว ลองเช็คสัญญาณเน็ตแล้วกด retry
- ข้อมูลที่ sync สำเร็จแล้วจะถูกลบออกจากรายการรอ sync
