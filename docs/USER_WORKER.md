## 1. การเข้าสู่ระบบ

### วิธีที่ 1: ผ่าน Web Browser

1. เปิดเบราว์เซอร์ (Chrome, Safari, Edge)
2. ไปที่ `http://admin.farm.localhost`
3. กรอกข้อมูลดังนี้:
   - **Username:** ชื่อผู้ใช้ที่ได้รับจาก Manager
   - **Password:** รหัสผ่านที่ได้รับ
4. คลิก **"เข้าสู่ระบบ"**

### วิธีที่ 2: ผ่าน LINE App (LIFF)

1. เปิด LINE และไปที่ **@Farm Tracking** (หรือลิงก์ที่ได้รับจาก Manager)
2. คลิก **"เปิดแอป"** หรือ **"เริ่มต้นใช้งาน"**
3. ระบบจะขออนุญาตเข้าถึงโปรไฟล์ LINE → คลิก **"อนุญาต"**
4. เข้าสู่ระบบอัตโนมัติ (ไม่ต้องกรอกรหัสผ่าน)

### 📌 เคล็ดลับ

> หากลืมรหัสผ่าน กดปุ่ม **"ลืมรหัสผ่าน?"** และทำตามขั้นตอนที่ปรากฏ

## 4. วิธีสแกน QR Code

### QR Code คืออะไร?

**QR Code** คือรหัสสี่เหลี่ยมที่ติดอยู่ที่แต่ละแปลงและต้นไม้ ใช้สำหรับระบุตัวตนของแปลง/พืชอย่างรวดเร็ว

### วิธีสแกน (ผ่าน LINE LIFF)

1. เปิด LINE ไปที่ **@Farm Tracking**
2. คลิกปุ่ม **"📷 สแกน QR"** ที่อยู่ด้านล่างหน้าจอ
3. อนุญาตให้กล้องทำงาน → คลิก **"อนุญาต"**
4. นำกล้องไปส่องที่ **QR Code** ที่ติดอยู่ที่แปลง/ต้นไม้

```
┌─────────────────────────────────────┐
│                                     │
│         ┌───────────┐               │
│         │ ▓▓▓▓▓▓▓▓▓ │               │
│         │ ▓ ██ ██ ▓ │               │
│         │ ▓ ██ ██ ▓ │  ← ส่องที่นี่  │
│         │ ▓▓▓▓▓▓▓▓▓ │               │
│         └───────────┘               │
│                                     │
│     📷 ส่อง QR Code ให้อยู่ในกรอบ     │
│                                     │
│     กดปุ่มชัตเตอร์หรือรอให้สแกนอัตโนมัติ │
│                                     │
└─────────────────────────────────────┘
```

### สิ่งที่ทำได้หลังสแกน

เมื่อสแกนสำเร็จ ระบบจะแสดง:

```
┌─────────────────────────────────────┐
│  ✅ พบข้อมูล!                        │
├─────────────────────────────────────┤
│                                     │
│  🌳 ต้นมะม่วง #12                    │
│  📍 แปลงที่ 3 - สวนมะม่วง            │
│  🌱 พันธุ์: มะม่วงน้ำดอกไม้          │
│  📅 ปลูกเมื่อ: 15 มกราคม 2567       │
│  📊 สถานะ: 🟢 สุขภาพดี              │
│                                     │
│  ─────────────────────────────      │
│  ดำเนินการ:                         │
│  [💧 บันทึกรดน้ำ]                   │
│  [🧪 บันทึกใส่ปุ๋ย]                  │
│  [⚠️ แจ้งปัญหา]                      │
│  [📋 ดูประวัติ]                      │
│                                     │
└─────────────────────────────────────┘
```

> 💡 **เคล็ดลับ:** หาก QR Code เสียหายหรือหลุด กด **"แจ้ง QR Code เสีย"** เพื่อแจ้งให้ Manager ทราบ

| 🐛 | **แมลง** | หนอน, เพลี้ย, ไร, มด |
| 💀 | **ต้นไม้ตาย** | ใบเหี่ยว, รากเน่า, ต้นแห้งตาย |

| ❌ ปิด (ไม่รับ) | ปัญหาถูกปฏิเสธ (ไม่ใช่ปัญหาจริง) |

## 6. ดูสถานะงาน

### 6.1 ติดตามงานของฉัน

ไปที่ **"งานของฉัน"** เพื่อดู:

- 📋 งานที่ได้รับมอบหมายทั้งหมด
- 📅 งานที่กำลังจะถึงกำหนด
- ✅ งานที่ทำเสร็จแล้ว

### 6.2 อัปเดตสถานะงาน

**วิธีที่ 1: จากหน้ารายละเอียดงาน**

1. เปิดรายละเอียดงาน
2. เปลี่ยนสถานะ:
   - **เริ่มทำ** → คลิกปุ่ม **"เริ่มทำงาน"**
   - **ทำเสร็จ** → คลิกปุ่ม **"ทำเสร็จแล้ว"**

**วิธีที่ 2: จากหน้ารายการ**

1. ที่หน้ารายการงาน ดูคอลัมน์ **"สถานะ"**
2. คลิกที่สถานะปัจจุบัน
3. เลือกสถานะใหม่จาก dropdown

### 6.3 สถานะงาน

| สถานะ | สี | ความหมาย |
|-------|-----|----------|
| ⏳ รอดำเนินการ | เทา | ยังไม่ได้เริ่มทำ |
| 🔄 กำลังทำ | น้ำเงิน | อยู่ระหว่างดำเนินการ |
| ✅ เสร็จสิ้น | เขียว | ทำเสร็จแล้ว |
| ❌ ยกเลิก | แดง | งานถูกยกเลิก |

### 6.4 ดูประวัติงาน

หากต้องการดูว่าเคยทำกิจกรรมอะไรกับแปลง/ต้นไม้นี้บ้าง:

1. สแกน QR Code หรือค้นหาจากรายการ
2. คลิก **"📋 ดูประวัติ"**
3. ระบบจะแสดงกิจกรรมทั้งหมดที่เคยบันทึก

```

│  📋 ประวัติกิจกรรม - ต้นมะม่วง #12         │

│  ├─ 💧 รดน้ำ 10 ลิตร (โดย: คุณวิน)         │
│  └─ ✅ งานเสร็จสมบูรณ์                     │
│                                             │
│  📅 24 มีนาคม 2569                         │
│  ├─ 🧪 ใส่ปุ๋ย NPK 100 กรัม               │
│  └─ 🔬 ตรวจสอบพบใบไหม้ → แจ้งปัญหาแล้ว    │
```

## ❓ คำถามที่พบบ่อย

**Q: หากไม่มีสัญญาณอินเทอร์เน็ตในฟาร์ม จะบันทึกข้อมูลได้ไหม?**
> A: สามารถบันทึกแบบ Offline ได้ เมื่อมีสัญญาณอินเทอร์เน็ต ระบบจะ Sync ข้อมูลอัตโนมัติ

**Q: สามารถแก้ไขกิจกรรมที่บันทึกผิดได้ไหม?**
> A: สามารถแก้ไขได้ภายใน 24 ชั่วโมง หลังจากนั้นต้องติดต่อ Manager

**Q: หาก QR Code ไม่สแกนอ่านไม่ได้?**
> A: สามารถพิมพ์ QR Code ใหม่ได้โดยกด "แจ้ง QR Code เสีย" หรือค้นหาจากรายการแทน
