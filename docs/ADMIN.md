## 1. Admin Panel Overview

### 🎛️ Admin Panel คืออะไร

Admin Panel คือหน้าจอจัดการระบบหลังบ้าน (Backend) ที่ให้ผู้ดูแลระบบ (Admin) สามารถจัดการข้อมูลทั้งหมดของระบบจัดการฟาร์มได้ ไม่ว่าจะเป็นผู้ใช้ ฟาร์ม โซน แปลง พืช และการตั้งค่าต่าง ๆ

### 🔐 วิธีเข้าสู่ Admin Panel

1. เปิดเบราว์เซอร์ (Chrome, Firefox, Safari, Edge)
2. ไปที่ URL: https://your-domain.com/admin
3. ล็อกอินด้วยบัญชี Admin

```
┌─────────────────────────────────────┐
│  🔐 เข้าสู่ระบบ Admin Panel       │
├─────────────────────────────────────┤
│  ชื่อผู้ใช้: [________________]   │
│  รหัสผ่าน:  [________________]   │
│                                    │
│         [ เข้าสู่ระบบ ]            │
└─────────────────────────────────────┘
```

### 📊 หน้าจอหลัก Dashboard

เมื่อล็อกอินสำเร็จ จะเข้าสู่หน้า Dashboard แสดง:
- จำนวนฟาร์มทั้งหมด
- จำนวนผู้ใช้ทั้งหมด
- จำนวนงานวันนี้
- จำนวนปัญหาที่รอดำเนินการ
- จำนวนแปลงทั้งหมด
- สถิติกิจกรรมเดือนนี้
- กราฟกิจกรรม 7 วันล่าสุด

### 🎯 Role และสิทธิ์การเข้าถึง

| Role | Dashboard | Users | Farms | Zones | Plots | Plants | Settings | Logs |
|------|-----------|-------|-------|-------|-------|--------|---------|------|
| super_admin | เต็ม | เต็ม | เต็ม | เต็ม | เต็ม | เต็ม | เต็ม | เต็ม |
| owner | ดู | เต็ม | เต็ม | เต็ม | เต็ม | เต็ม | เต็ม | ดู |
| manager | ดู | ดู | ดู | เต็ม | เต็ม | เต็ม | - | - |
| worker | ดู | - | - | - | - | - | - | - |

## 2. จัดการ Users (เพิ่ม, แก้ไข, ลบ)

### 👥 หน้าจอจัดการ Users

ไปที่เมนู "ผู้ใช้" เพื่อจัดการบัญชีผู้ใช้งานระบบ

**รายการผู้ใช้แสดง:** ชื่อ, บทบาท, ฟาร์ม, สถานะ, ปุ่มจัดการ

### ➕ เพิ่มผู้ใช้ใหม่

1. แตะปุ่ม "[+ เพิ่มผู้ใช้]"
2. กรอกข้อมูล:
   - ชื่อ-นามสกุล
   - อีเมล (ต้องไม่ซ้ำกับผู้ใช้อื่น)
   - เบอร์โทร
   - รหัสผ่าน + ยืนยันรหัสผ่าน
   - บทบาท (Role): super_admin, owner, manager, หรือ worker
   - ฟาร์มที่สังกัด (ไม่ต้องเลือก ถ้าเป็น super_admin)
   - หมายเหตุ (ถ้ามี)
3. แตะ "บันทึก"

### ✏️ แก้ไขผู้ใช้

1. แตะปุ่ม "[แก้ไข]" ในแถวของผู้ใช้ที่ต้องการ
2. แก้ไขข้อมูลที่ต้องการ
3. แตะ "บันทึก"

**สิ่งที่แก้ไขได้:**
- ชื่อ-นามสกุล
- อีเมล
- เบอร์โทร
- บทบาท (Role)
- ฟาร์มที่สังกัด
- สถานะ (เปิด/ระงับ)
- รหัสผ่าน (กดตั้งใหม่)

### 🗑️ ลบผู้ใช้

1. แตะปุ่ม "[ลบ]" ในแถวของผู้ใช้ที่ต้องการ
2. ระบบจะแสดงการยืนยัน พร้อมคำเตือน:
   - ผู้ใช้นี้จะไม่สามารถเข้าใช้งานได้อีก
   - ข้อมูลกิจกรรมที่บันทึกไว้จะยังคงอยู่
   - การกระทำนี้ไม่สามารถย้อนกลับได้
3. แตะ "ลบ" เพื่อยืนยัน

### 🔄 รีเซ็ตรหัสผ่าน

1. แก้ไขผู้ใช้
2. แตะปุ่ม "ตั้งรหัสผ่านใหม่"
3. ระบบจะส่งลิงก์ตั้งรหัสใหม่ไปที่อีเมลของผู้ใช้

## 3. จัดการ Roles (super_admin, owner, manager, worker)

### 🎭 Roles ในระบบ

ระบบมี 4 ระดับบทบาท (Roles):

| Role | คำอธิบาย | สิทธิ์หลัก |
|------|-----------|-----------|
| **super_admin** | ผู้ดูแลระบบสูงสุด | จัดการทุกอย่าง รวมถึง super_admin คนอื่น |
| **owner** | เจ้าของฟาร์ม | จัดการฟาร์มตัวเอง ผู้ใช้ในฟาร์ม ทุกอย่างยกเว้น logs |
| **manager** | ผู้จัดการฟาร์ม | จัดการ Zone, Plot, Plant, มอบหมายงาน |
| **worker** | ผู้ปฏิบัติงาน | ดู Dashboard, บันทึกกิจกรรมผ่าน LIFF App |

### 👑 Super Admin

ผู้ดูแลระบบสูงสุด มีสิทธิ์ทำทุกอย่างในระบบ

**สิทธิ์:**
- จัดการ Users ทุกคน (รวม super_admin คนอื่น)
- จัดการ Farms ทุกฟาร์ม
- จัดการ System Settings
- ดู Logs ทั้งหมด
- ตั้งค่า LIFF App
- สร้าง/ลบ super_admin คนอื่น

### 🏡 Owner

เจ้าของฟาร์ม มีสิทธิ์จัดการฟาร์มของตัวเองเท่านั้น

**สิทธิ์:**
- จัดการ Users ในฟาร์มตัวเอง
- จัดการ Zones, Plots, Plants ในฟาร์มตัวเอง
- ดู Dashboard ของฟาร์มตัวเอง
- ตั้งค่าพื้นฐานของฟาร์ม
- ดู Logs ของฟาร์มตัวเอง
- ไม่สามารถจัดการฟาร์มคนอื่น
- ไม่สามารถแก้ไข System Settings

### 📋 Manager

ผู้จัดการฟาร์ม มอบหมายโดย Owner

**สิทธิ์:**
- จัดการ Zones, Plots, Plants ในฟาร์มที่ได้รับมอบหมาย
- มอบหมายงานให้ worker
- ดู Dashboard ของฟาร์ม
- รายงานปัญหาที่พบ
- ไม่สามารถจัดการ Users
- ไม่สามารถตั้งค่าระบบ

### 👷 Worker

ผู้ปฏิบัติงานในฟาร์ม ใช้งานผ่าน LIFF App เป็นหลัก

**สิทธิ์:**
- บันทึกกิจกรรมผ่าน LIFF App
- รายงานปัญหาผ่าน LIFF App
- ดูงานที่ได้รับมอบหมาย
- ดู Dashboard เบื้องต้น
- ไม่สามารถเข้า Admin Panel (ยกเว้น Dashboard)

### 🔧 วิธีเปลี่ยน Role

1. ไปที่ "ผู้ใช้"
2. แตะ [แก้ไข] ที่ผู้ใช้ที่ต้องการ
3. เปลี่ยน "บทบาท" ใน dropdown
4. แตะ "บันทึก"

## 4. จัดการ Farms

### 🏡 หน้าจอจัดการ Farms

ไปที่เมนู "ฟาร์ม" เพื่อจัดการข้อมูลฟาร์ม

**รายการฟาร์มแสดง:** ชื่อฟาร์ม, เจ้าของ, พื้นที่, ปุ่มจัดการ

### ➕ เพิ่มฟาร์มใหม่

1. แตะปุ่ม "[+ เพิ่มฟาร์ม]"
2. กรอกข้อมูล:
   - ชื่อฟาร์ม
   - ที่อยู่
   - พื้นที่ (ไร่)
   - เบอร์ติดต่อ
   - เจ้าของ (เลือกจากรายการ Users ที่เป็น owner)
   - พิกัด GPS (ละติจูด, ลองจิจูด)
   - รายละเอียดเพิ่มเติม
3. แตะ "บันทึก"

### ✏️ แก้ไข/ดูรายละเอียดฟาร์ม

1. แตะปุ่ม "[แก้ไข]" หรือคลิกที่ชื่อฟาร์ม
2. Dashboard ของฟาร์มจะแสดง:
   - สถิติโซน, แปลง, พืช
   - กิจกรรมล่าสุด
   - ปัญหาที่ยังค้าง
   - งานที่กำลังดำเนินการ
   - รายงานสรุปประจำเดือน

### 🗑️ ลบฟาร์ม

1. แตะปุ่ม "[ลบ]" ในแถวของฟาร์มที่ต้องการ
2. ระบบจะเตือนว่า: การลบฟาร์มจะลบโซน, แปลง, พืช และข้อมูลทั้งหมดภายในฟาร์มด้วย
3. ยืนยันการลบ

## 5. จัดการ Zones

### 🗺️ Zone คืออะไร

Zone (โซน) คือพื้นที่แบ่งย่อยภายในฟาร์ม เช่น โซน A, โซน B, โซนสวนมะม่วง, โซนเก็บผลผลิต

### 📋 หน้าจอจัดการ Zones

ไปที่เมนู "โซน" เลือกฟาร์มที่ต้องการจัดการ

**รายการโซนแสดง:** ชื่อโซน, ประเภท, จำนวนแปลง, พื้นที่, ปุ่มจัดการ

### ➕ เพิ่มโซนใหม่

1. แตะปุ่ม "[+ เพิ่มโซน]"
2. กรอกข้อมูล:
   - ชื่อโซน
   - ประเภทโซน: สวนผลไม้, ไร่เพาะปลูก, โรงเรือน, โกดัง/คลัง, แหล่งน้ำ, อื่น ๆ
   - พื้นที่ (ไร่)
   - รายละเอียด
3. แตะ "บันทึก"

## 6. จัดการ Plots

### 🌱 Plot (แปลง) คืออะไร

Plot (แปลง) คือพื้นที่ปลูกพืชภายในโซน เช่น แปลง A-01, แปลง A-02

### 📋 หน้าจอจัดการ Plots

ไปที่เมนู "แปลง" เลือกฟาร์มและโซนที่ต้องการ

**รายการแปลงแสดง:** ชื่อแปลง, พืชที่ปลูก, จำนวนต้น, สถานะ, ปุ่มจัดการ

**สถานะแปลง:**
- ปกติ (พืชสุขภาพดี)
- เฝ้าระวัง (พบปัญหาเล็กน้อย)
- มีปัญหา (ต้องดูแลเร่งด่วน)
- ว่าง (ยังไม่ปลูก)

### ➕ เพิ่มแปลงใหม่

1. แตะปุ่ม "[+ เพิ่มแปลง]"
2. กรอกข้อมูล:
   - ชื่อแปลง
   - พืชที่ปลูก (เลือกจากรายการ หรือเพิ่มพืชใหม่)
   - จำนวนต้น
   - พื้นที่ (ตารางเมตร)
   - วันที่ปลูก
   - สถานะ
   - หมายเหตุ
3. แตะ "บันทึก"

## 7. จัดการ Plants

### 🌳 หน้าจอจัดการ Plants

ไปที่เมนู "พืช" เพื่อจัดการข้อมูลพืชที่ปลูกในฟาร์ม

**รายการพืชแสดง:** ชื่อพืช, ประเภท, จำนวนแปลงที่ปลูก, จำนวนต้นรวม

### ➕ เพิ่มพืชใหม่

1. แตะปุ่ม "[+ เพิ่มพืช]"
2. กรอกข้อมูล:
   - ชื่อพืช
   - ประเภท: ไม้ผล/ไม้ยืนต้น, พืชไร่/พืชล้มลุก, ผัก, ไม้ดอก, สมุนไพร, อื่น ๆ
   - อายุเฉลี่ย (เดือน)
   - ความต้องการน้ำ: น้อย, ปานกลาง, มาก
   - ความต้องการปุ๋ย: น้อย, ปานกลาง, มาก
   - รายละเอียดพิเศษ
3. แตะ "บันทึก"

### 🌿 ดูรายละเอียดพืช

เมื่อคลิกที่ชื่อพืช จะเห็น:
- จำนวนแปลงที่ปลูกพืชนี้
- จำนวนต้นรวม
- กิจกรรมล่าสุดที่เกี่ยวกับพืชนี้
- ปัญหาที่พบ
- ข้อมูลการดูแล (รดน้ำ, ใส่ปุ๋ย, พ่นยา ล่าสุด)

## 8. QR Code Management

### 📱 QR Code ในระบบ

ระบบสร้าง QR Code สำหรับระบุตำแหน่งในฟาร์ม:

| ประเภท QR | ข้อมูลที่เก็บ | การใช้งาน |
|-----------|-------------|-----------|
| **Farm QR** | ID ฟาร์ม, ชื่อ, พิกัด | สแกนเพื่อเช็คอินฟาร์ม |
| **Zone QR** | ID โซน, ID ฟาร์ม, ชื่อ | สแกนเพื่อระบุโซน |
| **Plot QR** | ID แปลง, ID โซน, ชื่อแปลง | สแกนเพื่อระบุแปลง |
| **Plant QR** | ID พืช, ID แปลง, ชื่อพืช | สแกนเพื่อระบุต้นเฉพาะ |
| **Activity QR** | ID กิจกรรม, ตำแหน่ง | สแกนเพื่อบันทึกกิจกรรม |

### 🖨️ สร้างและพิมพ์ QR Code

1. ไปที่เมนู "QR Code"
2. เลือกประเภท: ฟาร์ม, โซน, แปลง, พืช
3. เลือกรายการที่ต้องการสร้าง QR
4. ระบบจะสร้าง QR Code พร้อมข้อมูล
5. ดาวน์โหลดเป็นรูปภาพ (PNG, SVG) หรือพิมพ์โดยตรง

### 🖨️ พิมพ์ QR Code หลายรายการ

1. เลือกรายการที่ต้องการ (ติ๊กเครื่องหมาย)
2. แตะปุ่ม "พิมพ์ QR ที่เลือก"
3. เลือกขนาดกระดาษและจำนวนช่อง
4. ระบบจะสร้าง PDF สำหรับพิมพ์

### 📋 QR Code Template

แนะนำขนาด QR Code:
- QR Code บนป้าย: ขนาด 5x5 ซม. ขึ้นไป
- QR Code บนไม้กาบ: ขนาด 2x2 ซม.
- QR Code ในโปสเตอร์: ขนาด 10x10 ซม. ขึ้นไป

## 9. System Settings

### ⚙️ หน้าจอ System Settings

ไปที่เมนู "ตั้งค่า" เพื่อจัดการการตั้งค่าระบบ (เฉพาะ super_admin และ owner)

### 🏷️ ตั้งค่าทั่วไป

- ชื่อระบบ / ชื่อองค์กร
- โลโก้
- ข้อมูลติดต่อ
- เขตเวลา (Timezone)
- ภาษาเริ่มต้น (ไทย/อังกฤษ)

### 📱 ตั้งค่า LIFF App

- LIFF Channel ID
- LIFF Channel Secret
- URL ของ LIFF App
- ข้อความต้อนรับ
- Quick Actions ที่แสดง

### 🔔 ตั้งค่าการแจ้งเตือน

- เปิด/ปิดการแจ้งเตือน LINE
- กลุ่มที่ส่งการแจ้งเตือน (Owner, Manager)
- ประเภทการแจ้งเตือน:
  - งานใหม่มอบหมาย
  - งานใกล้ถึงกำหนด
  - งานเลยกำหนด
  - รายงานปัญหาใหม่
  - รายงานปัญหาด่วน

### 📊 ตั้งค่าการบันทึก

- ระยะเวลาเก็บข้อมูลกิจกรรม (เดือน)
- ระยะเวลาเก็บ Logs (เดือน)
- ระยะเวลาเก็บรูปภาพ (เดือน)

### 🔐 ตั้งค่าความปลอดภัย

- เปลี่ยนรหัสผ่าน Admin

## 10. Logs และ Monitoring

### 📋 Logs ในระบบ

ไปที่เมนู "Logs" เพื่อดูประวัติการใช้งานระบบ (เฉพาะ super_admin และ owner)

### 📊 ประเภท Logs

| ประเภท | ข้อมูลที่บันทึก |
|--------|----------------|
| **Activity Logs** | กิจกรรมที่บันทึก (รดน้ำ, ใส่ปุ๋ย, ฯลฯ) |
| **System Logs** | การเข้า/ออกระบบ, การแก้ไขข้อมูล |
| **Error Logs** | ข้อผิดพลาดที่เกิดขึ้นในระบบ |
| **Sync Logs** | การ sync ข้อมูล LIFF App |

### 🔍 วิธีดู Logs

1. เลือกประเภท Logs
2. กรองข้อมูลตาม:
   - ช่วงวันที่
   - ผู้ใช้
   - ฟาร์ม/โซน/แปลง
   - ประเภทกิจกรรม
3. ค้นหาด้วยคำค้นหา
4. ดูรายละเอียดโดยคลิกที่รายการ

### 📤 Export Logs

1. เลือกช่วงวันที่และตัวกรอง
2. แตะปุ่ม "Export"
3. เลือกรูปแบบ: CSV, Excel, PDF
4. ระบบจะสร้างไฟล์และดาวน์โหลด

### 📈 Dashboard Monitoring

หน้า Dashboard มีข้อมูลสรุป:

- **การใช้งานระบบ:** จำนวนผู้ใช้งานวันนี้, สัปดาห์นี้, เดือนนี้
- **กิจกรรม:** สรุปกิจกรรมตามประเภท
- **ปัญหา:** สรุปปัญหาตามสถานะ
- **LIFF App:** จำนวนการใช้งาน, อัตรา sync สำเร็จ
- **แปลงที่มีปัญหา:** แสดงรายการแปลงที่มีสถานะ "มีปัญหา" หรือ "เฝ้าระวัง"
